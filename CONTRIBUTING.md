# Contributing to AuditCheck AI

Thank you for your interest in contributing to AuditCheck AI! We are building a transparent healthcare pricing engine, and we welcome community support to expand our dataset and improve the user interface.

**Important:** To maintain system stability and security, we have strict guidelines on which parts of the codebase can be modified. Please read the [Contribution Rules](Contribution-Rules.md) before starting any work.

## üöÄ Quick Start Guide

1.  **Fork the Repository**: Click the "Fork" button at the top right of this page.
2.  **Clone Your Fork**:
    ```bash
    git clone [https://github.com/YOUR-USERNAME/AuditCheck.AI-Powered-Billing-Companion.git](https://github.com/YOUR-USERNAME/AuditCheck.AI-Powered-Billing-Companion.git)
    cd AuditCheck.AI-Powered-Billing-Companion
    ```
3.  **Create a Branch**:
    ```bash
    git checkout -b feature/add-new-hospital-data
    ```
4.  **Make Changes**: Follow the rules in `Contribution-Rules.md`.
5.  **Test Your Changes**: Ensure the app runs locally with `npm start`.
6.  **Submit a Pull Request**: Push to your fork and submit a PR to our `main` branch.
